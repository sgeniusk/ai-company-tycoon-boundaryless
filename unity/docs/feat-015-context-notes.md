# feat-015 컨텍스트 노트 — 지분·자본 사다리: 개업 인터뷰, 주식, 융자, 상장, 부자 순위 (설계 초안)

사용자 방향(2026-06-11 밤) — "무조건 차고(0성) 시작이 도파민 — 빠른 시작과 개업 시작 방식 둘을 두자. 개업하면서 인터뷰를 해 — '투자를 좀 할래? 그럼 주식을 나눠주겠어'. 직원 3명 고용에 초기 주주처럼 주식 분배. 은행 대출 + 투자 권유 요소. 경영정보에 주식 근황. 4성쯤 되면 상장 이벤트 — 그럼 세계 부자 순위가 전광판에."

## 0. 핵심 루프 (도파민 설계)
가난한 차고 0성 → 지분을 아껴 키운 회사 가치가 숫자로 자람 → 4성 **상장 빅 모먼트**(종 울리기) → 전광판이 기업 랭킹에서 **세계 부자 순위**로 확장 → 1위 등반. 초반 선택(지분을 팔아 빠르게 vs 아껴서 크게)이 엔드게임 순위를 결정한다.

이 시스템은 React에 없다 — **Unity 오리지널 콘텐츠 1호**. 정본 데이터를 새로 만든다(equity_events.json 등, 루트 data/에 additive).

## 1. 시작 방식 — 빠른 시작 vs 개업 스토리
- 더보기 새 게임에 모드 2개: **빠른 시작**(현행 그대로 — 회귀 없음) / **개업 스토리**(차고 0성 고정 + 인터뷰 시퀀스).
- 0성 표기 — 차고 프로토타입 단계를 크레스트 "0★"으로(현 1★). 표기만, 스테이지 데이터 불변.
- **개업 인터뷰** — 시작 시 카드 3~4장의 선택 시퀀스(기자 인터뷰 톤).
  1. 엔젤 투자 제안 — "투자해도 될까?" 선택지: 거절(지분 100%) / 소액($8K, 지분 -10%) / 통 큰 투자($20K, 지분 -25%).
  2. 공동창업 권유 — 첫 직원에게 지분 5%씩 분배하면 영입비 무료+충성(이탈 없음) vs 월급만.
  3. 은행 — 개업 대출 제안(아래 §3).
  - 전부 시드와 무관한 플레이어 선택 — 리플레이 다양성은 선택 조합에서.

## 2. 지분(Equity) 상태 — 세이브 v6
- `GameModel.FounderEquity` (0~100, 시작 100) + `Shareholders` 목록(이름·지분·유형 angel/cofounder/public).
- **회사 가치 평가(파생, 저장 안 함)** — `밸류에이션 = 월매출 * 24 + 이용자 * 2 + 현금` (단순 공식, 밸런스 패스에서 조정). 내 자산 = 밸류에이션 * 지분% + 현금성 보너스.
- 경영 탭 **경영정보 카드** — 회사 가치, 내 지분 %, 내 자산, 주주 목록. 월 정산마다 갱신 — 자라는 게 보인다.

## 3. 은행 융자 + 투자 권유
- **대출** — 경영 탭 액션. 한도 = 신뢰·월매출 기반(예: 월매출 * 6). 월 이자(원금의 1.5%/월)가 고정비에 추가. 조기 상환 가능. `GameModel.LoanPrincipal` (v6).
- **투자 권유(런 중)** — 성장 마일스톤(2성 승급, 이용자 1만 등)마다 시리즈 A/B 투자 제안 이벤트: 현금 큰 거 + 지분 -X%. 거절 가능 — 지분을 아끼는 플레이가 성립해야 한다.

## 4. 상장(IPO) — 4성 빅 모먼트
- 조건 — 4성(ai_platform_giant) 도달 + 월매출/신뢰 기준 + 미상장.
- 연출 — 전용 모달(종 치기, SpawnCelebration 최대치): 공모 지분 결정(10/20/30%) → 공모가 = 밸류에이션 기반 → 대형 현금 유입.
- 상장 후 — **주가**가 월별로 움직인다(시드 파생 + 실적 연동: 매출 성장·신뢰·세계 이벤트 반영). 경영정보 카드가 주가 차트 톤으로 진화. 주주 압박 이벤트 풀 추가(분기 실적 기대 등).
- 엔딩 연계 — 멀티 엔딩(24종)에 지분/상장 조건 엔딩 추가 후보(예: 지분 90%+ 상장 = "고독한 제왕").

## 5. 세계 부자 순위 — 전광판 확장
- 상장 후 LED 전광판이 기업 랭킹과 **부자 순위**를 교차 표시(마키 활용).
- 경쟁사 CEO 자산 = 경쟁사 score 파생(MarketService.CompetitorStates 재사용 — 신규 시스템 불필요).
- 내 순위 상승 시 토스트("세계 부자 #37 — 12계단 상승!").
