# Alpha v0.14.7 제작 보고서 — 카드 보상 편향 표시

작성일: 2026-05-16

## 요약

v0.14.7은 v0.14.6에서 만든 “연간 지시가 카드 보상 후보를 편향한다”는 숨은 효과를 덱 보상 화면에 표시한 버전이다. 이제 카드 보상 3택1이 떠 있을 때, 현재 연간 운영 지시가 어떤 카드 태그를 밀어주는지 바로 읽을 수 있다.

## 구현 내용

- `getAnnualDirectiveRewardBiasSummary` 읽기 모델 추가.
- 활성 연간 지시의 보상 편향 태그를 한글 라벨로 변환.
- 카드 보상 패널에 지시명과 편향 설명을 표시하는 `reward-bias-strip` 추가.
- 덱빌딩 테스트에 UI용 요약 모델 검증 추가.

## 하네스 관점 리뷰

- UX Agent: 숨은 산식이 보상 선택 직전에 노출되어 선택 이유를 이해하기 쉬워졌다.
- Deck System Engineer: 카드 보상 후보가 왜 이런 방향으로 떴는지 설명할 최소 표면을 확보했다.
- Retention / LTV Agent: 연간 지시 선택이 다음 보상까지 이어진다는 피드백이 생겨 한 해를 더 굴릴 이유가 강해졌다.
- Solo Dev Scope Agent: 새 화면 없이 기존 보상 패널에 짧은 정보 스트립을 추가해 구현 범위를 통제했다.
