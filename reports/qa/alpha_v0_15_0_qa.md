# Alpha v0.15.0 QA 보고서

작성일: 2026-05-16

## 범위

이번 QA는 연간 운영 지시가 제품, 연구, 경쟁 대응 추천으로 확장되는지 확인한다.

## 실행 명령

- `npm test -- src/game/annual-strategy-advisor.test.ts src/game/qa-scenarios.test.ts`: 통과, 23 tests.
- `npm run harness:gate`: 통과, 153 tests, 데이터 검증 통과, 프로덕션 빌드 통과.
- Headless Chrome screenshot QA: `http://127.0.0.1:5178/?scenario=annual-strategy` 1366x768 렌더링 확인.

## 이슈

- P0/P1 없음.
- P2: 전략실 추천을 클릭해 메뉴를 이동하는 상호작용은 아직 없다.
- P2: 추천 점수 분포를 장기 시뮬레이션 로그로 기록하지는 않는다.

## 결론

v0.15.0은 자동 테스트와 브라우저 스크린샷 기준으로 연간 지시를 다음 행동 추천으로 확장했다. 회사 화면에서 플레이어가 제품, 연구, 경쟁 대응 후보를 함께 읽을 수 있다.
