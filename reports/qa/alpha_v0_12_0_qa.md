## 브라우저 QA 메모

이 세션에서는 Browser 플러그인 자동 조작 도구가 노출되지 않았고, Node REPL 환경에도 Playwright가 설치되어 있지 않았다. 따라서 스크린샷 기반 시각 QA는 수행하지 못했다. 대신 `?scenario=deck` 상태는 자동 시나리오 테스트와 프로덕션 빌드로 검증했다.

## 잔여 리스크

- 브라우저 시각 QA는 다음 세션에서 실제 화면으로 추가 확인이 필요하다.
- 퍼즐 직접 선택 UI가 없어 현재는 상위 4개 이슈를 자동 해결한다.
- 메타 해금 버튼은 “새 런”을 즉시 시작하므로, 다음 버전에서 확인 모달이나 결과 화면을 붙이는 편이 안전하다.
