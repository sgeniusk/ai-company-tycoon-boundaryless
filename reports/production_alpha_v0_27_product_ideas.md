# v0.27-alpha 제작 보고서 — 제품 아이디어 조합실과 리뉴얼 출시 기반

작성일: 2026-05-17

## 목표

이번 단계의 목표는 제품 개발이 단순히 “새 제품 목록을 하나씩 찍어내는 구조”에 머물지 않게 만드는 것이다. 게임발전국의 조합 재미를 AI 회사 소재에 맞게 확장하고, 이미 출시한 대표 제품을 계속 키우는 리뉴얼 출시 기반을 마련했다.

## 예시 조합

- `전기차 + 차량 + 초개인화`
  - 운전자 성향에 맞춰 주행감과 실내 경험이 바뀌는 AI 전기차
- `커피 프랜차이즈 + 프랜차이즈 시스템 + 완전 무인`
  - 무인 바리스타 매장 운영망
- `웹소설 + 에이전트 스튜디오 + 생성형 IP`
  - 세계관과 캐릭터를 자동 확장하는 연재 스튜디오

## 하네스 에이전트 리뷰

### Executive Producer Agent

- 판정: 통과
- 사유: 제품 개발의 장기 재미가 강화됐다. 새 제품만 무한히 늘리는 위험을 줄이고, 대표 제품을 키우는 방향이 생겼다.

### Game Designer Agent

- 판정: 통과
- 사유: 소재/타입/파격 옵션 3축은 게임발전국식 조합 재미를 AI 회사에 맞게 확장한다. 엉뚱한 산업 확장도 자연스럽게 받아낼 수 있다.

### Systems Architect Agent

- 판정: 통과
- 사유: 조합 소재는 JSON 데이터로 분리했고, 조합 결과는 순수 함수로 계산한다. UI는 계산 결과를 표시만 한다.

### QA Agent

- 판정: 통과
- 확인: 5,184개 조합 전체 생성 테스트, 레이아웃 계약, 데이터 검증, 전체 하네스 게이트 통과.

### Balance Agent

- 판정: 통과
- 사유: 아직 실제 경제 효과는 직접 적용하지 않고 궁합 점수와 비용 후보로만 보여준다. 다음 단계에서 밸런스 적용 폭을 조절하기 쉽다.

### UX Agent

- 판정: 통과
- 사유: 제품 메뉴에서 바로 조합 결과를 확인할 수 있다. 다만 모바일에서는 내부 스크롤로 확인해야 하므로 다음 UI 패스에서 접힘/요약 표시를 검토한다.

### Retention / LTV Agent

- 판정: 통과
- 사유: “이번 런에서는 어떤 이상한 회사를 만들까?”라는 반복 동기를 강화한다.

### Shareability Agent

- 판정: 통과
- 사유: 전기차, 커피, 웹소설, 우주 산업 같은 조합 결과가 스크린샷과 플레이 후기 소재가 된다.

### Solo Dev Scope Agent

- 판정: 통과
- 사유: 콘텐츠 폭은 데이터로 늘렸고, 실제 출시/경제 연결은 다음 단계로 미뤄 범위를 통제했다.

## 백로그

- P1: 조합 결과를 실제 신규 제품 프로젝트 생성으로 연결한다.
- P1: 기존 제품 리뉴얼 후보를 실제 개발 프로젝트로 연결한다.
- P2: 제품명 자동 생성 규칙을 더 풍부하게 만든다.
- P2: 조합 궁합 점수가 출시 리뷰와 카드 보상에 영향을 주게 한다.
- P3: 모바일 제품 메뉴에서 조합실을 접을 수 있게 한다.
