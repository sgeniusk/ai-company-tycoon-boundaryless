# v0.34.1-alpha 제작 보고서 — 게임 화면 조작 레이어

작성일: 2026-05-17

## 목표

v0.34에서 첫 화면 HUD는 조였지만, 메뉴와 사무실 조작이 아직 웹 콘솔처럼 보였다. 이번 패스는 플레이어가 “어디를 눌러야 하는지”와 “경쟁사가 지금 움직이고 있다”를 첫 화면에서 느끼게 하는 데 집중했다.

## 이번 구현

- 우측 메뉴를 핵심 4개와 보조 4개 클러스터로 나눴다.
- 모바일에서는 `회사`, `제품`, `덱`, `에이전트`, `더보기` 5탭을 먼저 보여준다.
- 사무실 안에 `고용`, `개발`, `전략`, `꾸미기` 액션 슬롯을 추가했다.
- 경쟁사 현재 이벤트나 모멘텀이 가장 높은 경쟁사를 `경쟁 속보` 배너로 표시한다.
- v0.34.1 레이아웃 계약 테스트를 추가했다.

## 하네스 에이전트 검토

- UX Agent: P1 해결. 핵심 메뉴와 보조 메뉴의 위계가 생겼고, 모바일에서 첫 탭 선택지가 명확해졌다.
- Game Designer Agent: P1 해결. 사무실 클릭 슬롯이 제품/인력/전략/공간 성장 루프를 플레이필드에 연결한다.
- Shareability Agent: P2 개선. 경쟁 속보 배너가 스크린샷에서 “무슨 일이 났는지”를 더 잘 보여준다.
- Solo Dev Scope Agent: P2 양호. 기존 메뉴와 경쟁사 데이터만 재배치해 코드/에셋 부담을 낮췄다.

## 남은 작업

- 카드 사용 시 짧은 피드백 연출을 추가한다.
- 사무실 액션 슬롯을 실제 직원/장식 위치 슬롯과 더 세밀하게 연결한다.
- 모바일 회사/덱 패널을 접힘 요약형으로 더 압축한다.
- 20인 페르소나 테스트를 v0.34.1 기준으로 다시 돌린다.
