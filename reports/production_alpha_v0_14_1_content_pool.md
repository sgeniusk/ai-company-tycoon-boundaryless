# Alpha v0.14.1 제작 보고서 — 콘텐츠 풀 확장

작성일: 2026-05-16

## 요약

이번 버전은 새로운 대형 시스템보다 10년 캠페인을 버틸 기본 콘텐츠를 채우는 작업이다. 앞으로 제품, 사무실, 직원 조합을 늘릴 때 빈 슬롯이 생기지 않도록 사람 직원, AI 에이전트, 로봇 인력, 아이템, 사무실 사물 훅을 확장했다.

## 구현 내용

- 고용 후보 22종 준비.
- 사람 직원 6종, AI 에이전트 13종, 로봇 인력 3종으로 명시 분류.
- 신규 사람 직원: 차고 PM 인턴, 야간 서버 당직, 현장 영업 담당, 조직문화 매니저.
- 신규 AI 에이전트: 벤치마크 분석 에이전트, 시나리오 시뮬레이터, 산업 디자인 AI.
- 신규 로봇 인력: 물류 로봇 클러스터, 휴머노이드 랩 어시스턴트.
- 아이템 37종 준비.
- 신규 아이템 카테고리: 복지, 제조, 하드웨어, 모빌리티.
- 사무실 오브젝트 21종과 아이템 아이콘 훅 18종 준비.
- 각 신규 인력에 선호 아이템과 외형 키워드 연결.

## 하네스 관점

- Game Designer: 초반 차고 인력, 중반 검증 인력, 후반 산업 확장 인력의 단계감이 생겼다.
- Systems Architect: 새 콘텐츠는 JSON 데이터와 에셋 매니페스트 중심으로 추가되어 UI와 시뮬레이션 구조를 흔들지 않는다.
- UX Agent: 아이템 카테고리 표시명을 추가해 상점에서 새 카테고리가 원문으로 노출되지 않는다.
- Solo Dev Scope Agent: 실제 그래픽 제작 전, 필요한 오브젝트와 아이콘 연결만 먼저 준비했다.
- Retention/LTV Agent: 장기 캠페인 중 “다음에 고용하고 싶은 후보”와 “다음에 사고 싶은 사물”의 수가 늘었다.

## 검증

- `npm test -- src/game/content.test.ts src/game/asset-manifest.test.ts`: 통과, 12 tests.
- `npm run validate:data`: 통과.
- `npm run harness:gate`: 통과, 126 tests, 데이터 검증 통과, 프로덕션 빌드 통과.

## 배포 정책

이번 버전은 콘텐츠 증분이므로 Vercel 배포는 하지 않았다. 앞으로도 큰 버전업이나 공개 확인이 필요한 시점에만 Vercel에 배포한다.

## 다음 액션

1. 신규 인력과 아이템을 제품군/산업 확장 루프에 더 직접적으로 연결한다.
2. 상점과 고용 화면에 카테고리 필터 또는 추천 묶음을 추가한다.
3. 신규 사무실 오브젝트가 실제 화면에서 더 다양하게 보이도록 배치 밀도와 위치 규칙을 개선한다.
