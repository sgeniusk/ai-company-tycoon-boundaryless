# v0.41-alpha 제작 보고서 — 사무실 픽셀 시뮬레이션 1차

작성일: 2026-05-19

## 목표

시스템이 복잡해진 느낌을 줄이기 위해 회사 운영 상태를 숫자 패널보다 먼저 “사무실 안에서 벌어지는 일”로 보이게 만든다. AI Town처럼 캐릭터가 공간 안에 존재한다는 감각을 프로토타입 레벨에서 먼저 확보한다.

## 구현

- `data/office_scene.json` 추가: 지휘 데스크, 연산 베이, 면접 코너, 런칭 무대, 라운지, 로봇 베이, 칩 랩, 쇼룸, 이동 동선, 운영 스크린을 데이터화.
- `getOfficeScenePlan()` 추가: 사무실 단계, 활성 구획, 프로젝트 배치, 직원 상태를 화면 오브젝트와 액터 상태로 변환.
- 메인 사무실 화면 추가: 픽셀 그리드, 구획 오브젝트, 사람/AI/로봇 액터, 작업/휴식/리텐션 경고 말풍선.
- `office-visuals` QA 시나리오 추가: 구획과 로봇 액터가 함께 보이는 상태로 바로 진입.
