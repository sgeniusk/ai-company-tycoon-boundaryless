# Alpha v0.12.4 제작 보고서 — AI 모델 출발과 경계 없는 산업 확장

작성일: 2026-05-15

## 요약

이번 버전은 게임의 장기 판타지를 더 선명하게 바꿨다. 회사는 이제 처음부터 “AI 앱 회사”가 아니라 작은 파운데이션 모델을 만드는 회사로 시작한다. 이후 반도체, 차량, 로봇, 커피 프랜차이즈, 장난감 같은 이상할 정도로 넓은 산업으로 확장할 수 있다.

또한 개발 퍼즐은 핵심 진행 장치가 아니라 선택형 `개발 이슈 대응`으로 낮췄다. 기본 개발력은 직원, 에이전트, 나중에는 로봇형 개발 인력이 만든다.

## 구현 내용

- 초반 출시 가능 제품 `파운데이션 모델 v0` 추가.
- 신규 도메인 추가: 파운데이션 모델, AI 반도체, 모빌리티, 로봇, 기묘한 소비 산업, 장난감.
- 신규 제품 추가: 프론티어 추론 모델, AI 학습 칩, 자율주행 AI 스택, 창고 로봇 플릿, AI 바리스타 카페 체인, 상상력 장난감 엔진.
- 신규 능력치 `로봇공학` 추가.
- 로봇공학 Lv.1 이후 고용 가능한 `팩토리 로봇 유닛` 추가.
- 초기 경쟁사를 5개에서 8개 이상으로 확장.
- 12개월차 신규 강력 경쟁사: 오토노바 모터스, 브루체인.
- 24개월차 신규 강력 경쟁사: 토이클라우드, 아이언오라클.
- 경쟁사 데이터에 `entry_month`, `rival_tier`, `entry_announcement`를 추가.
- 제품 화면에 산업 확장 지도 추가.
- 경쟁 화면에 앞으로 등장할 경쟁사 목록 추가.
- `?scenario=rivals` QA 시나리오 추가.

## 에이전트 리뷰

### Executive Producer Agent

- P0/P1 없음.
- 장기 판타지가 “여러 AI 앱을 만든다”에서 “모델을 만든 회사가 산업을 침범한다”로 커졌다.

### Game Designer Agent

- P0/P1 없음.
- 커피 체인과 장난감 같은 의외의 산업이 AI 소재의 웃긴 확장성을 살린다.
- P2: 실제 플레이에서 산업 확장이 너무 멀게만 느껴지지 않도록 중간 보상과 티저가 필요하다.

### Systems Architect Agent

- P0/P1 없음.
- 경쟁사 등장은 데이터 필드로 열었고, 월 진행 시 자동으로 상태에 추가된다.
- P2: 산업별 제품 태그와 에이전트 시너지는 별도 룰 모듈로 분리하는 편이 좋다.

### QA Agent

- P0/P1 없음.
- 새 데이터, 신규 경쟁사 등장, 로봇 인력 잠금/해금 조건을 테스트로 고정했다.

### Balance Agent

- P0/P1 없음.
- 후반 제품은 요구 능력과 비용이 높아 초반 경제를 즉시 깨지 않는다.
- P2: 12개월 경쟁사 등장 시 시장 점유율 충격이 너무 크지 않은지 장기 시뮬레이션이 필요하다.

### UX Agent

- P0/P1 없음.
- 제품 화면의 확장 지도가 장기 목표를 보여준다.
- P2: 제품 수가 늘어나면서 필터/도메인 탭이 필요해질 가능성이 커졌다.

### Solo Dev Scope Agent

- P0/P1 없음.
- 그래픽 없이 데이터와 작은 UI 블록만으로 장기 확장감을 만들었다.
- P2: 다음 콘텐츠 확장은 카드/제품/경쟁사를 템플릿으로 늘리는 방식이 안전하다.
