# v0.15.4-alpha 제작 보고서 — 회사 승급 트랙

작성일: 2026-05-17

## 목표

이번 단계의 목표는 플레이어가 현재 회사가 몇 성급인지뿐 아니라, 다음 회사 단계까지 무엇이 부족한지 즉시 알 수 있게 만드는 것이다. 기존에는 연간 심사와 10년 캠페인 진행률은 보였지만, “시골 차고에서 다음 회사로 올라가는 조건”이 게임 화면 안에서 충분히 직접적이지 않았다.

## 구현 내용

- `getCompanyStageProgress`를 추가해 현재 회사 단계, 다음 단계, 조건별 현재값/목표값/완료 여부를 계산한다.
- 회사 단계 조건 계산을 공통 함수로 정리해 별 등급 계산과 승급 트랙이 같은 기준을 사용하게 했다.
- 메인 사무실 옆 회사 단계 카드에 `다음 승급` 미니 체크리스트를 추가했다.
- 회사 현황 메뉴에 `회사 승급 트랙` 패널을 추가해 다음 단계 조건, 진행률, 현재 별 등급을 표시했다.
- 최종 회사 단계에 도달한 경우에는 남은 목표가 10년 엔딩 점수와 시장 지배력임을 안내한다.

## 하네스 에이전트 검토

### Executive Producer Agent

- P0/P1 없음.
- 다음 승급 조건이 화면에 고정되어 10분 MVP의 목표성이 더 분명해졌다.

### Game Designer Agent

- P0/P1 없음.
- `차고 프로토타입 → 시드 스타트업` 흐름이 명확해져 카이로소프트식 승급 목표와 잘 맞는다.
- 향후 지역 이전, 경쟁사 시즌, 산업 확장을 이 승급 트랙과 더 강하게 묶을 수 있다.

### Systems Architect Agent

- P0/P1 없음.
- 회사 단계 진행도 계산을 `campaign.ts`에 모아 데이터 기반 회사 단계와 UI가 같은 값을 바라보게 했다.

### UX Agent

- P0/P1 없음.
- 메인 화면의 미니 패널은 즉시 목표를 알려주고, 회사 메뉴의 상세 패널은 조건을 더 크게 보여준다.
- 화면 밀도는 조금 높아졌으므로 v0.16에서 우측 패널 압축과 툴팁 분리가 필요하다.

### Retention / LTV Agent

- P0/P1 없음.
- 플레이어가 “다음 별 등급까지만 해보자”는 짧은 목표를 갖기 쉬워졌다.

### Solo Dev Scope Agent

- P0/P1 없음.
- 새 데이터 파일 없이 기존 `company_stages.json`를 재사용해 유지보수 부담이 낮다.

## 남은 과제

- 승급 순간에 별 등급 상승 연출과 보상 팝업을 추가한다.
- 승급 트랙과 지역 이전 추천을 더 직접적으로 연결한다.
- v0.15.5에서 경쟁사 시즌 이벤트와 승급 목표를 묶어 시장이 살아 움직이는 느낌을 강화한다.
