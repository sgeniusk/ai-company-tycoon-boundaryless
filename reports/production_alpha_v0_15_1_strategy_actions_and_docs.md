# Alpha v0.15.1 제작 보고서 — 전략실 액션화와 문서 정리

작성일: 2026-05-16

## 요약

v0.15.1은 `연간 전략실`을 읽는 패널에서 행동으로 이어지는 패널로 바꾸는 작은 기능 개선과, 앞으로 개발을 빠르게 보기 위한 문서 정리를 함께 진행한 버전이다.

## 구현 내용

- 전략실 추천 모델에 액션 후보를 추가했다.
- 제품 추천은 `products`, 연구 추천은 `research`, 경쟁 대응 추천은 `competition` 메뉴로 연결된다.
- 회사 화면의 전략실 패널에 추천 액션 버튼을 추가했다.
- 로드맵을 과거 로그 중심에서 앞으로의 계획 중심으로 재작성했다.
- 변경 기록을 최근 주요 버전 중심으로 축약했다.

## 하네스 관점 리뷰

- UX Agent: 추천이 “읽는 정보”에서 “바로 움직이는 버튼”으로 바뀌어 다음 행동이 분명해졌다.
- Retention / LTV Agent: 연간 지시 선택 이후 다음 메뉴 이동이 쉬워져 한 해 더 진행할 동기가 선명해졌다.
- Solo Dev Scope Agent: 추천 하이라이트나 딥링크 상태까지 확장하지 않고 메뉴 이동 버튼까지만 구현해 범위를 통제했다.
- Executive Producer Agent: 문서가 앞으로의 개발 판단에 쓰이도록 짧아졌다.
