# v0.20-alpha 밸런스 보고서

작성일: 2026-05-17

## 하네스 결과

`evaluateAlphaReadiness` 기준으로 다음 게이트를 평가한다.

- `commercial_paths`: 모든 성장 경로의 10개월 상용 루프
- `ten_year_campaign`: 120개월 캠페인 완주
- `integrity`: 상태 무결성
- `ending`: 10년 엔딩 랭크

## 해석

- 10개월 상용 루프와 10년 캠페인이 같은 코드 하네스 안에서 검증된다.
- 시즌 과제 미해결 압박은 경쟁사 모멘텀을 올려 시장이 정지하지 않게 한다.
- 시즌 과제 완료 보상은 신뢰, 이용자, 데이터로 제한해 과도한 현금 눈덩이를 피했다.

## 조정 후보

- 시즌 압박 모멘텀 `+2`는 첫 값으로 적당하지만, 실제 플레이에서 과도하면 `+1`로 낮춘다.
- 시즌 과제 완료 보상 `신뢰 +1, 이용자 +80, 데이터 +2`는 반복 보상이므로 v0.21에서 월별 중복 체감 여부를 확인한다.
- 후반 산업 확장 목표는 명확해졌지만, 실제 후반 산업 제품까지 도달하는 수동 플레이 난도는 별도 테스트가 필요하다.

## 판정

v0.20 기준 밸런스는 알파 플레이테스트에 투입 가능하다. 정식 베타 전에는 최소 20인 페르소나 테스트와 장기 시뮬레이션 로그를 다시 비교해야 한다.
