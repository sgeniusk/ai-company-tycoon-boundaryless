# 프로덕션 보고서 — v0.12.1-alpha 카드와 퍼즐의 직접 조작

작성일: 2026-05-15

## 목표

v0.12.0에서 만든 덱/퍼즐 기반을 실제 조작감으로 끌어올렸다. 이번 버전의 핵심은 플레이어가 퍼즐 칸을 직접 고르고, 카드 사용이 다음 퍼즐의 점수와 난이도에 영향을 주도록 연결하는 것이다.

## 구현 내용

- 개발 퍼즐 칸을 직접 선택하는 UI로 변경했다.
- 선택 개수와 선택 제한을 화면에 표시했다.
- `선택 이슈 해결` 버튼은 해결 조건이 맞을 때만 활성화되게 했다.
- 전략 카드에 퍼즐 보정 효과를 추가했다.
  - `puzzle_score_bonus`: 다음 퍼즐 점수 증가.
  - `puzzle_difficulty_delta`: 관련 이슈 난이도 감소.
  - `puzzle_tile_limit`: 한 번에 해결 가능한 칸 수 증가.
- 카드가 만든 보정은 현재 프로젝트에 1회 적용되고, 퍼즐 해결 후 소모된다.
- 퍼즐 결과에 적용된 카드 이름을 기록하고 UI에 표시한다.

## P2 백로그

- 출시 후 카드 보상 3택1이 필요하다.
- 퍼즐 선택이 제품 출시 리뷰에 더 직접적으로 연결되어야 한다.
- 카드 보정 효과를 카드 상세에 더 짧고 명확하게 보여줄 필요가 있다.
- 브라우저 시각 QA에서 모바일 보드 터치 영역을 확인해야 한다.
