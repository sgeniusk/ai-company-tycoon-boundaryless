# 디자인 방향 — post-알파 비전 & 도파민 (2026-05-29 확정)

작성: Claude Code · 출처: 2026-05-29 디자인 리뷰 세션 (사용자 결정)
관련: `reports/v0_61_game_design_review_slides.html` (리뷰 덱), `feature_list.json` (v0.62~)

이 문서는 v0.61 공개 알파 직후의 디자인 리뷰에서 사용자가 내린 **확정 결정**과 **북극성 비전**을 보존한다. v0.62 페이오프 연출과 post-1.0 대형 시스템(로그라이크·태그엔진·멀티엔딩)의 기준점.

## 1. 북극성 (취지)

> 카이로소프트보다 **훨씬 세밀한 픽셀아트**, **열린 도전** 같은 느낌, **로그라이크 요소**를 담아 **계속 반복해서 플레이**하게 만든다. 더 많은 **이벤트와 풍성한 콘텐츠**.

핵심 KPI는 리플레이성 — "한 판 더"를 만드는 것. 이를 위해 (a) 판마다 다른 변주, (b) 발견의 즐거움, (c) 터지는 페이오프, (d) 긴장-해소가 맞물려야 한다.

## 2. 확정 결정 (디자인 리뷰 A/B/C)

- **A. "무궁무진 조합" 엔진 → 태그 기반 파생 엔진.** 시너지/조합을 손으로 열거하지 않고 도메인·제품·카드의 태그 관계에서 계산 → 콘텐츠 추가 시 조합이 자동 증식. (도파민 = 발견.) post-1.0.
- **B. 리플레이성/내러티브 → 멀티 엔딩·회사 정체성 분기(1순위) + 로그라이크 런 모디파이어(크게 확장).** post-1.0.
- **C. 베타 게이트 → 아트 + 플레이테스트 병렬.** 아트 투자와 재미 검증은 별개로 본다. 아트 방향은 현 픽셀/아이소 유지하되 **해상도 상향**.

## 3. 로그라이크 런 모디파이어 (B 확장 — post-1.0)

런 = **환경 × 시장 × 조건 × 주인공 능력**의 조합. 시작 시 뽑고, **매년/전개될수록 변수가 계속** 들어온다.

| 축 | 예시 | 기존 자산 |
|---|---|---|
| 🌍 시작 도시 | 서울 · 도쿄 · 뉴욕 · 샌프란시스코 · 텍사스 … | `data/company_locations.json` (지역·인재풀·비용 모디파이어) **이미 존재** → 시작점 선택으로 승격 |
| 🌐 세계관/타임라인 | "알파고 패배로 LLM 발전 정체" · "비트코인 지배로 GPU 초고가" · "오픈소스 천국" · "규제 강국" | 신규 `run_modifiers` (compute 비용·연구 속도·신뢰 동역학 등을 시프트) |
| 📊 시장 상황 | 호황/불황, 특정 도메인 과열 | `campaign_shocks` 효과 모델 재사용 |
| 🧑 창업자 특성 | 엔지니어/마케터/연구자 출신 등 시작 보너스·약점 | 신규 (소규모 데이터 + 시작 상태 변형) |
| 📅 연중 변수 | 매년 호재/악재 "세계 이벤트" 리빌 | `data/campaign_shocks.json` (10년 페이싱 쇼크) **이미 존재** → 풍성한 이벤트 시스템으로 확장 |

요지 — **새로 짓는 게 아니라 기존 데이터(company_locations + campaign_shocks + roguelite 런 상태) 위에 모디파이어 레이어를 얹는 일.** "풍성한 이벤트/콘텐츠"는 campaign_shocks + 세계 이벤트 + 태그 파생 조합의 곱으로 확보.

## 4. 도파민 프레임워크 — 4개 레버

도파민 = **발견 × 변주 × 페이오프 연출 × 긴장-해소.**

1. **발견 (Discovery)** — 태그 엔진이 예상 못한 조합 생성 → "처음 발견!" + **도감(수집)**. (결정 A)
2. **변주 (Variety)** — 로그라이크 런 셋업 = "이번 판은 뭐지?" 기대 + 연중 이벤트. (결정 B)
3. **페이오프 연출 (Payoff juice)** — 큰 조합/마일스톤이 터질 때 이름 박힌 풀스크린 이벤트 + 숫자 카운트업 + 플레어. **→ v0.62에서 즉시 착수 (§5-safe, 기존 모먼트 증폭).**
4. **긴장-해소 (Tension→relief)** — 아슬아슬 연간 심사 통과·막판 라이벌 역전. **일부는 난이도 조정으로 해결** (사용자 메모) → 별도 밸런스 패스 + near-miss 연출.

구체 모먼트 — 대박 조합 발동 연출 · 조합 발견 셀러브레이션 · 런 시작 "세계 뽑기" 리빌 · 연중 "세계 이벤트" 룰렛 · 마일스톤 팡파레(첫 100만·첫 물리 산업·AGI 임계·보스 라이벌 격파) · near-miss 강조.
