## 1. 현재 목표

v0.56 pre-art 슬라이스는 닫혔고, v0.57 코어 재미 개선(`v0.57-alpha-core-fun-polish`)도 9개 `#N` 폴리시 + 4개 P1 폴리시 커밋으로 닫혔다. 다음 마일스톤은 미선정 상태로 대기 중이다. 후보는 `v0.58-alpha-market-season-strength`(시장/경쟁사 시즌 강화), P2 AGY 5회 + 실제 사람 5회 블라인드 트랙(최종 아트 게이트 해제용), v0.57 P2 모바일 백로그다.

## 3. 현재 결정

`v0.55-alpha`는 **최종 아트 교체 대비 스크린샷 QA 하네스 완료**로 닫는다.

완료로 보는 이유:

- 최종 아트가 아직 없어도, 들어오는 즉시 검수할 수 있는 임포트/스크린샷 경로가 있다.
- 데스크톱 1366×768과 모바일 390×844 기준선 PNG가 남아 있다.
- 모바일 command HUD 오른쪽 잘림은 해결됐다.

별도 분리:

- 실제 최종 캐릭터/오브젝트/배경 원본 제작 및 교체는 `docs/ART_INTAKE.md`의 Art Intake 작업으로 분리한다.
- 최종 아트 부재는 `v0.56` 플레이테스트 슬라이스를 막는 P0가 아니다.
- 최종 그래픽 에셋 투입은 `npm run qa:art-gate`가 `최종 그래픽 에셋 투입: 가능`을 보여준 뒤 시작한다. 이 명령은 내부적으로 `npm run qa:blind-summary`와 `npm run qa:blind-issues`를 다시 실행해 `실제 세션: 5/5`, `열린 P0: 0`, `P0 미기록: 0`, `상태 미인정: 0`, `증거 미기록: 0`, `아트 투입 판정: 가능`, `P0 큐: 0`을 합산한다. `열린 P1`과 `P1 큐`는 별도 튜닝 후보로 남겨도 된다.
