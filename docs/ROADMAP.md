# 로드맵 — AI Company Tycoon: Boundaryless

작성일: 2026-05-15

## 현재 상태

현재 버전은 `v0.12.4-alpha`다. 프로젝트 방향은 **카이로소프트식 AI 회사 경영 + 로그라이트 덱빌딩 + 선택형 개발 이슈 대응**으로 조정했다. 그래픽 에셋을 제외한 핵심 프로토타입은 다음 기반을 갖췄다.

- 첫 제품 개발과 출시.
- 첫 AI 모델 제품인 파운데이션 모델 v0.
- 출시 평가, 성장 경로 선택, 경쟁사 압박.
- 10개월 MVP 진행도.
- 업적/런 목표와 보상.
- 시작 덱, 손패, 드로우/버림 더미.
- 전략 카드 사용으로 개발 진행도/완성도 변화.
- 직원/에이전트 배치를 기본 개발력으로 삼는 제품 개발.
- 선택형 개발 이슈 대응.
- 카드가 이슈 대응 난이도, 점수, 선택 칸 수에 영향을 주는 카드-이슈 시너지.
- 출시 후 카드 보상 3택1.
- 덱 편집 토큰으로 카드 제거/강화.
- 반도체, 모빌리티, 로봇, 커피 체인, 장난감 산업으로 이어지는 경계 없는 확장.
- 제품 개발 시작 전 직접 고르는 직원 배치.
- 직원 조합이 개발 기간보다 완성도와 출시 리뷰에 더 크게 영향을 주는 출시 산식.
- 보유 아이템을 확인하는 인벤토리 패널.
- 실제 고용 직원과 프로젝트 보드가 보이는 사무실 화면.
- 로봇공학 연구 후 고용 가능한 로봇형 개발 인력.
- 초반 8개 경쟁사와 12개월/24개월 신규 강력 경쟁사 등장 기반.
- 창업 통찰, 메타 해금, 새 런 시작.
- 성장 경로 월간 효과.
- 제품 업그레이드.
- 저장/불러오기와 깨진 저장 복구.
- 런 결과 랭크와 다음 행동 추천.
- 자동 밸런스 시뮬레이션 하네스.

## 개발 원칙

- 사용자에게 보여 주는 보고서와 로드맵은 한국어로 작성한다.
- 한 번 개선할 때마다 버전을 붙이고 변경 기록을 남긴다.
- 플레이어가 느끼는 재미를 먼저 고정하고, 그래픽은 그 다음에 붙인다.
- 매 버전은 테스트, 데이터 검증, 빌드를 통과해야 한다.
- 큰 기능은 반드시 QA 시나리오나 시뮬레이션 하네스로 반복 검증한다.

## 단기 계획

### v0.12.1-alpha — 카드와 퍼즐의 직접 조작

상태: 완료.

목표: 덱빌딩과 개발 퍼즐을 “보이는 장식”이 아니라 제품 개발의 핵심 손맛으로 만든다.

구현:

- 퍼즐 칸을 플레이어가 직접 선택할 수 있게 한다.
- 카드가 퍼즐 난이도, 선택 가능 칸, 점수 배율에 영향을 주게 한다.
- 퍼즐 점수를 출시 리뷰와 월간 성과에 더 명확히 연결한다.
- `?scenario=deck` 브라우저 QA를 정식 체크리스트에 넣는다.

완료 기준:

- 플레이어가 제품 개발 중 카드 1장과 퍼즐 1회를 직접 사용할 수 있다.
- 퍼즐 결과가 진행도, 완성도, 출시 평가 중 최소 2개 지표에 반영된다.
- 카드-퍼즐 상호작용 테스트가 있다.
- `npm run harness:gate` 통과.

### v0.12.2-alpha — 완성도 중심 출시와 직원 배치

상태: 완료.

목표: 제품 개발을 “기다리는 막대”가 아니라, 직원 배치·카드·퍼즐·아이템이 완성도를 만드는 플레이로 바꾼다.

구현:

- 제품 개발 시작 전 투입 에이전트를 직접 선택한다.
- 선택 팀에 따라 예상 완성도, 리뷰 등급, 리뷰 점수, 월 완성도 상승을 보여준다.
- 프로젝트 월간 진행도는 안정화하고, 팀 조합은 완성도와 출시 리뷰에 더 크게 반영한다.
- 인벤토리 패널로 구매한 아이템과 장착 대기 장비를 확인한다.
- 사무실 화면에 실제 고용 직원과 프로젝트 보드를 표시한다.
- `?scenario=staffing` 브라우저 QA를 추가한다.

완료 기준:

- 직원 미선택 상태에서는 프로젝트 시작이 막힌다.
- 선택한 직원만 프로젝트에 배치된다.
- 서로 다른 팀 조합이 같은 기간 안에서도 출시 평가를 다르게 만든다.
- `npm run harness:gate` 통과.

상태: 완료.

목표: 매 런이 다른 회사가 되도록 카드 보상, 제거, 강화 선택을 추가한다.

- 카드 희귀도, 비용, 효과, 강화 여부를 덱 메뉴에서 표시한다.

- `?scenario=reward` 브라우저 QA를 추가한다.

완료 기준:

- 첫 출시 후 카드 보상을 선택할 수 있다.
- 카드 제거/강화가 저장/불러오기에 보존된다.

- `npm run harness:gate` 통과.

### v0.12.4-alpha — AI 모델 출발과 경계 없는 산업 확장

상태: 완료.

목표: AI 회사가 처음에는 모델을 만들고, 이후 반도체·차량·로봇·커피 프랜차이즈·장난감 같은 이상할 정도로 넓은 산업으로 뻗는 판타지를 데이터와 UI에 심는다.

구현:

- `파운데이션 모델 v0`를 초반 출시 가능 제품으로 추가한다.
- 반도체, 모빌리티, 로봇, 기묘한 소비 산업, 장난감 도메인을 추가한다.
- AI 학습 칩, 자율주행 AI 스택, 창고 로봇 플릿, AI 바리스타 카페 체인, 상상력 장난감 엔진을 추가한다.
- 로봇공학 능력과 로봇형 개발 인력을 추가한다.
- 초기 경쟁사를 8개 이상으로 늘리고, 12개월/24개월에 강력한 신규 경쟁사가 등장하게 한다.
- 제품 화면에 산업 확장 지도를 표시한다.
- 경쟁 화면에 예정 경쟁사 등장을 표시한다.
- `?scenario=rivals` 브라우저 QA를 추가한다.

완료 기준:

- 새 게임에서 파운데이션 모델 v0를 출시할 수 있다.
- 반도체, 모빌리티, 로봇, 기묘한 소비 산업, 장난감 제품이 데이터로 존재한다.
- 12개월차에 오토노바 모터스와 브루체인이 등장한다.
- 로봇공학 Lv.1 이후 로봇형 개발 인력을 고용할 수 있다.
- 데이터 검증과 관련 테스트가 통과한다.
